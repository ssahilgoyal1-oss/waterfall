## 📌 Task Overview
This task involves creating a **Waterfall Development Model** for a new startup company's product using Lucid Chart.

The Waterfall Model is a **sequential software development methodology** where each phase is completed fully before moving to the next one.

## 🔄 Waterfall Model Phases

| Phase | Name | Description |
|-------|------|-------------|
| 1️⃣ | Requirements Gathering | Collect all stakeholder needs and create SRS document |
| 2️⃣ | System Design | Plan architecture, database, UI mockups & tech stack |
| 3️⃣ | Implementation | Write code and build features based on design |
| 4️⃣ | Testing & QA | Unit testing, integration testing, and UAT |
| 5️⃣ | Deployment | Release product to production/cloud servers |
| 6️⃣ | Maintenance | Bug fixes, patches, and future updates |
